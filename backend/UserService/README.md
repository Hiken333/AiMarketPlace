# UserService - Сервис управления пользователями

Микросервис для управления пользователями маркетплейса AI моделей.

## Назначение

Сервис отвечает за регистрацию, аутентификацию, управление профилями пользователей и их правами доступа на маркетплейсе.

## Технологический стек

- **Backend**: Java 21, Spring Boot 3.5.3
- **Framework**: Spring Boot Starter

## Структура проекта

```
src/main/
├── java/              # Java код сервиса
└── resources/         # Конфигурационные файлы
    └── application.yml
```

## Основной функционал

- Регистрация и аутентификация пользователей
- Управление профилями пользователей
- Управление ролями и правами доступа
- Хранение данных пользователей
