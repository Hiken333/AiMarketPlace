# ReviewService - Сервис отзывов

Микросервис для управления отзывами и рейтингами AI моделей на маркетплейсе.

## Назначение

Сервис обрабатывает отзывы покупателей о AI моделях, управляет рейтингами и модерацией отзывов.

## Технологический стек

- **Backend**: Node.js
- **Framework**: Express 4.16.1
- **Middleware**: cookie-parser, morgan, debug

## Структура проекта

```
ReviewService/
├── app.js             # Главный файл приложения
├── routes/            # API маршруты
│   └── ...
├── bin/              # Скрипты запуска
│   └── www
└── package.json
```

## Основной функционал

- Создание и управление отзывами
- Система рейтингов
- Модерация отзывов
- Агрегация рейтингов продуктов
