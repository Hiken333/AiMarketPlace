# AnalyticsService - Сервис аналитики

Микросервис для сбора и анализа данных о продажах и использовании AI моделей на маркетплейсе.

## Назначение

Сервис собирает метрики использования AI моделей, анализирует продажи, формирует отчеты и статистику для администраторов и пользователей маркетплейса.

## Технологический стек

- **Backend**: Java 21, Spring Boot 3.5.3
- **Framework**: Spring Boot Starter

## Структура проекта

```
src/main/
├── java/              # Java код сервиса
└── resources/         # Конфигурационные файлы
    └── application.yml
```

## Основной функционал

- Сбор метрик использования AI моделей
- Анализ продаж и транзакций
- Формирование отчетов и статистики
- Агрегация данных для дашбордов
