# PaymentService - Сервис обработки платежей

Микросервис для обработки платежей и транзакций на маркетплейсе AI моделей.

## Назначение

Сервис обрабатывает платежи за покупку AI моделей, управляет транзакциями и интегрируется с платежными системами.

## Технологический стек

- **Backend**: Java 21, Spring Boot 3.5.3
- **Framework**: Spring Boot Starter

## Структура проекта

```
src/main/
├── java/              # Java код сервиса
└── resources/         # Конфигурационные файлы
    └── application.yml
```

## Основной функционал

- Обработка платежей
- Интеграция с платежными системами
- Управление транзакциями
- Обработка возвратов и возмещений
