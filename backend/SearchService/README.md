# SearchService - Сервис поиска

Микросервис для поиска AI моделей и продуктов на маркетплейсе.

## Назначение

Сервис предоставляет функционал поиска по каталогу AI моделей с фильтрацией, сортировкой и ранжированием результатов.

## Технологический стек

- **Backend**: Node.js
- **Framework**: Express 4.16.1
- **Middleware**: cookie-parser, morgan, debug

## Структура проекта

```
SearchService/
├── app.js             # Главный файл приложения
├── routes/            # API маршруты
│   └── ...
├── bin/              # Скрипты запуска
│   └── www
└── package.json
```

## Основной функционал

- Полнотекстовый поиск по продуктам
- Фильтрация и сортировка результатов
- Ранжирование результатов поиска
- Поисковые подсказки и автодополнение
