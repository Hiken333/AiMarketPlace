# FileService - Сервис управления файлами

Микросервис для загрузки, хранения и управления файлами на маркетплейсе AI моделей.

## Назначение

Сервис обрабатывает загрузку AI моделей, изображений, документации и других файлов, связанных с продуктами маркетплейса.

## Технологический стек

- **Backend**: Node.js
- **Framework**: Express 4.16.1
- **Middleware**: cookie-parser, morgan, debug

## Структура проекта

```
FileService/
├── app.js             # Главный файл приложения
├── routes/            # API маршруты
│   └── ...
├── bin/              # Скрипты запуска
│   └── www
└── package.json
```

## Основной функционал

- Загрузка и хранение файлов
- Управление файлами AI моделей
- Обработка изображений и документации
- Предоставление доступа к файлам
