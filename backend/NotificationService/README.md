# NotificationService - Сервис уведомлений

Микросервис для отправки уведомлений пользователям маркетплейса AI моделей.

## Назначение

Сервис отправляет уведомления о событиях: новых заказах, сообщениях, обновлениях продуктов и других важных событиях.

## Технологический стек

- **Backend**: Node.js
- **Framework**: Express 4.16.1
- **Middleware**: cookie-parser, morgan, debug

## Структура проекта

```
NotificationService/
├── app.js             # Главный файл приложения
├── routes/            # API маршруты
│   └── ...
├── bin/              # Скрипты запуска
│   └── www
└── package.json
```

## Основной функционал

- Отправка email уведомлений
- Push уведомления
- SMS уведомления (опционально)
- Управление подписками на уведомления
