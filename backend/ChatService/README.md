# ChatService - Сервис чата

Микросервис для обработки сообщений и чатов между пользователями маркетплейса.

## Назначение

Сервис обеспечивает коммуникацию между продавцами и покупателями AI моделей, обработку сообщений и уведомлений.

## Технологический стек

- **Backend**: Node.js
- **Framework**: Express 4.16.1
- **Middleware**: cookie-parser, morgan, debug

## Структура проекта

```
ChatService/
├── app.js             # Главный файл приложения
├── routes/            # API маршруты
│   └── ...
├── bin/              # Скрипты запуска
│   └── www
└── package.json
```

## Основной функционал

- Обработка сообщений между пользователями
- Управление чатами
- Уведомления о новых сообщениях
- История переписки
