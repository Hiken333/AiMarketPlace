# OrderService - Сервис управления заказами

Микросервис для управления заказами на маркетплейсе AI моделей.

## Назначение

Сервис обрабатывает создание, управление и отслеживание заказов на покупку AI моделей и связанных услуг.

## Технологический стек

- **Backend**: Java 21, Spring Boot 3.5.3
- **Framework**: Spring Boot Starter

## Структура проекта

```
src/main/
├── java/              # Java код сервиса
└── resources/         # Конфигурационные файлы
    └── application.yml
```

## Основной функционал

- Создание и управление заказами
- Отслеживание статусов заказов
- Обработка заказов на AI модели
- История заказов пользователей
