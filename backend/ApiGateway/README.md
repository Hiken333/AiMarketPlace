# ApiGateway - API Gateway

Точка входа для всех запросов к микросервисам маркетплейса AI моделей.

## Назначение

API Gateway маршрутизирует запросы к соответствующим микросервисам, обеспечивает единую точку входа и может выполнять функции балансировки нагрузки и аутентификации.

## Технологический стек

- **Backend**: Node.js
- **Framework**: Express 4.16.1
- **Middleware**: cookie-parser, morgan, debug

## Структура проекта

```
ApiGateway/
├── app.js             # Главный файл приложения
├── routes/            # Маршруты
│   ├── index.js
│   └── users.js
├── bin/              # Скрипты запуска
│   └── www
└── package.json
```

## Основной функционал

- Маршрутизация запросов к микросервисам
- Единая точка входа для API
- Балансировка нагрузки
- Логирование запросов
