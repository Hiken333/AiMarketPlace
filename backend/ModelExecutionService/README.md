# ModelExecutionService - Сервис выполнения моделей

Микросервис для выполнения и тестирования AI моделей на маркетплейсе.

## Назначение

Сервис предоставляет возможность выполнения AI моделей, тестирования их работы и получения результатов для демонстрации покупателям.

## Технологический стек

- **Backend**: Node.js
- **Framework**: Express 4.16.1
- **Middleware**: cookie-parser, morgan, debug

## Структура проекта

```
ModelExecutionService/
├── app.js             # Главный файл приложения
├── routes/            # API маршруты
│   └── ...
├── bin/              # Скрипты запуска
│   └── www
└── package.json
```

## Основной функционал

- Выполнение AI моделей
- Тестирование моделей
- Обработка входных данных
- Возврат результатов выполнения
