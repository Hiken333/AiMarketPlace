# ProductService - Сервис управления продуктами

Микросервис для управления AI моделями и продуктами на маркетплейсе.

## Назначение

Сервис управляет каталогом AI моделей, их описаниями, ценами, доступностью и метаданными.

## Технологический стек

- **Backend**: Java 21, Spring Boot 3.5.3
- **Framework**: Spring Boot Starter

## Структура проекта

```
src/main/
├── java/              # Java код сервиса
└── resources/         # Конфигурационные файлы
    └── application.yml
```

## Основной функционал

- Управление каталогом AI моделей
- CRUD операции с продуктами
- Управление ценами и доступностью
- Категоризация и тегирование продуктов
