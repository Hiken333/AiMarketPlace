# LoggingService - Сервис логирования

Микросервис для централизованного логирования и мониторинга событий в системе маркетплейса.

## Назначение

Сервис собирает, хранит и анализирует логи всех микросервисов маркетплейса для отладки, мониторинга и аудита.

## Технологический стек

- **Backend**: Java 21, Spring Boot 3.5.3
- **Framework**: Spring Boot Starter

## Структура проекта

```
src/main/
├── java/              # Java код сервиса
└── resources/         # Конфигурационные файлы
    └── application.yml
```

## Основной функционал

- Централизованное логирование
- Хранение и поиск логов
- Анализ и агрегация логов
- Мониторинг событий системы
